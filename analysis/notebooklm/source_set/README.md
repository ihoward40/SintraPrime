This directory contains the complete, hashed source set provided to NotebookLM
for read-only analysis.

Upload Procedure:
1) Run the exporter to populate files/ and source_set.json.
2) Verify source_set.json.sha256.
3) Upload only files/ to NotebookLM.
4) In NotebookLM, export the “Source list” and save it alongside this folder
   (do not modify source_set.json).

Notes:
- Outputs generated by NotebookLM are analytical aids only and are not evidence.
- Do not add or remove files without regenerating the source set and hash.
