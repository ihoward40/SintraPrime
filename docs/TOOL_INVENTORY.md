# SINTRAPRIME TOOL INVENTORY PACK (HIGH-VALUE, LOW-SPRAWL)

## Core (must-have)

- Notion (SoT)
- Make.com (complex workflows)
- Zapier (simple triggers)
- Gmail + logging pattern to Notion
- Google Drive/Trust Vault storage convention
- Run standards (RunID + manifest + approval-by-hash)

## Legal drafting (assistive; human review required)

- Clearbrief (cite-to-source workflow)
- Gavel (document automation templates)

Principle: add tools only when they reduce friction, not for collecting tools.

## Creative

- Canva (brand templates, resizing)
- VEED (caption accuracy, editing)
- InVideo (auto script-to-video when needed)
- Image generation for abstract designs (not real-person identification)

## Data

- Airtable (if Notion DB limits you)
- Looker Studio (quick dashboards)

## Coding

- GitHub Copilot (code generation + review discipline)
