PURPOSE:
This file documents how SintraPrime connects to ElevenLabs for mythic multi-voice speech.
No live API keys are stored here. All sensitive values go ONLY in local `.env` files.

------------------------------
1. REQUIRED SERVICES
------------------------------
- ElevenLabs account with:
  - Custom voice models for:
    - isiah (primary)
    - oracle
    - guardian
    - scribe
    - scholar
    - judge
    - trickster
    - dragon
    - angel

- (Optional) OpenAI / other LLM:
  - Used for generating the TEXT that gets spoken.
  - API keys are stored in `.env` on local machine or server.

------------------------------
2. ENVIRONMENT VARIABLES (.env)
------------------------------
Create a file:
  - SintraPrime_Mythic_Speech/config/.env

DO NOT SHARE THIS FILE.

Example contents (template):

ELEVEN_API_KEY=YOUR_ELEVENLABS_API_KEY_HERE
OPENAI_API_KEY=YOUR_OPENAI_KEY_HERE

# Optional: Slack
SLACK_BOT_TOKEN=YOUR_SLACK_BOT_TOKEN_HERE
SLACK_DEFAULT_CHANNEL=#all-ikesolutions

------------------------------
3. VOICE MAP (voices.json)
------------------------------
Create a file:
  - SintraPrime_Mythic_Speech/config/voices.json

Use ElevenLabs Voice IDs (safe to share), e.g.:

{
  "isiah":   "ELEVEN_VOICE_ID_FOR_ISIAH",
  "oracle":  "ELEVEN_VOICE_ID_FOR_BRICK_CITY_ORACLE",
  "guardian":"ELEVEN_VOICE_ID_FOR_VAULT_GUARDIAN",
  "scribe":  "ELEVEN_VOICE_ID_FOR_SOVEREIGN_SCRIBE",
  "scholar": "ELEVEN_VOICE_ID_FOR_STREET_SCHOLAR",
  "judge":   "ELEVEN_VOICE_ID_FOR_JUDGE_INVISIBLE",
  "trickster":"ELEVEN_VOICE_ID_FOR_TRICKSTER_AUDITOR",
  "dragon":  "ELEVEN_VOICE_ID_FOR_DRAGON_DUE_PROCESS",
  "angel":   "ELEVEN_VOICE_ID_FOR_ANGEL_REMEDY"
}

You can send this file to others or to ChatGPT. It does NOT expose your account.

------------------------------
4. CORE ENDPOINT (REFERENCE)
------------------------------
Base URL:
  https://api.elevenlabs.io/v1/text-to-speech/{voice_id}

Headers:
  Accept: audio/mpeg
  Content-Type: application/json
  xi-api-key: (from ELEVEN_API_KEY in .env)

Example JSON body:

{
  "text": "Sample text for SintraPrime Agent.",
  "model_id": "eleven_multilingual_v2",
  "voice_settings": {
    "stability": 0.5,
    "similarity_boost": 0.75
  }
}
