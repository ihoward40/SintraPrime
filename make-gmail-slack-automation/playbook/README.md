# Playbook (PDF)

Target file for incident binders:

- `playbook/Gmail-Slack-Alerts-Deterministic-Runbook.pdf`

## How to generate

Fastest path:

1. Paste the markdown from `docs/gmail-slack-label-gated-runbook.md` into Google
	Docs or Notion
2. Ensure page size is standard letter
3. Export as PDF
4. Save it into this folder with the exact filename above

This repo intentionally does not include a generated PDF by default.
