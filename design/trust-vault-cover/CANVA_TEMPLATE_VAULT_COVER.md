# Canva Template — Trust Vault Cover (Black‑Gold)

This is a designer-ready build sheet for Canva (or Figma/Affinity/Photoshop). It creates a “restricted archive / arcane codex” cover for:

- **THE HOWARD TRUST NAVIGATOR**
- **SINTRAPRIME MYTHIC SPEECH ENGINE**
- **MASTER CONFIGURATION DOSSIER — VAULT EDITION**

No secrets are included; only titles and placeholders.

---

## A) Print Master (Letter)

### Canvas
- **Size:** 8.5 in × 11 in (Letter)
- **DPI:** 300
- **Bleed:** 0.125 in on all sides
- **Safe margin:** 0.35 in inside trim

### Grid
- Create guides (from top-left, in inches):
  - **Vertical guides:** 0.35, 4.25, 8.15
  - **Horizontal guides:** 0.35, 1.15, 4.40, 9.40, 10.65

Meaning:
- 0.35" = safe margin
- 4.25" = center
- 1.15" = top title block zone bottom
- 4.40" = sigil zone centerline
- 9.40" = lower banner zone start
- 10.65" = footer zone baseline

---

## B) Color Palette (Black‑Gold)

Use these consistently:
- **Obsidian Black:** `#000000`
- **Gold Foil (Primary):** `#D4AF37`
- **Warm Gold (Glow):** `#BA9B57`
- **Pale Gold (Subtitle):** `#E1CC87`
- **Deep Gold (Accents):** `#C6A861`

Opacity defaults:
- Dust/specks: 10–15%
- Glow: 12–18%
- Divider lines: 75–90%

---

## C) Typography (Canva fonts)

Pick one pairing (A recommended):

**Pairing A (Recommended, widely available):**
- Title: **Cinzel** (ALL CAPS)
- Subtitles: **Cinzel** or **Cormorant Garamond**
- Metadata blocks: **Cormorant Garamond**

**Pairing B (More classical):**
- Title: **EB Garamond** (Tracking +20)
- Subtitles: **Cormorant Garamond**

Tracking:
- Title: +10 to +40
- Subtitle line 2 ("DOSSIER — VAULT EDITION"): +80 to +140

---

## D) Layer Stack (Top → Bottom)

### 1) Background Base
- Rectangle: Full page
- Fill: `#000000`

### 2) Texture Overlay (choose one)
- Place a black marble / smoked vellum texture image
- Opacity: 10–15%
- Blend: **Overlay** or **Multiply**

### 3) Gold Radial Glow
- Circle (large), centered at page center (4.25", 5.5")
- Fill: `#BA9B57`
- Opacity: 12–18%
- Blur: heavy (Canva blur slider high)

Goal: “throne-room aura,” not a spotlight.

### 4) Decorative Border
- Thin rectangle outline:
  - Position: 0.20" inset from trim edge
  - Stroke: 2 px equivalent
  - Color: `#C6A861` at 70% opacity
- Optional second inner border:
  - Position: 0.35" inset
  - Stroke: 1 px
  - Color: `#C6A861` at 55% opacity

### 5) Title Block (Top)
Place centered text:

**(5.1) Primary Title**
- Text: `THE HOWARD TRUST NAVIGATOR`
- Font: Cinzel
- Size: 54 pt (range 48–64)
- Color: `#D4AF37`
- Position (centered): Y ≈ 0.78" from top

**(5.2) Divider Ornament**
- Text: `────────── ❂ ──────────`
- Font: Cinzel or any serif
- Size: 16–18 pt
- Color: `#C6A861`
- Opacity: 80%
- Position: Y ≈ 1.05"

**(5.3) Subtitle 1**
- Text: `SINTRAPRIME MYTHIC SPEECH ENGINE`
- Font: Cinzel
- Size: 30 pt
- Color: `#E1CC87`
- Position: Y ≈ 1.28"

**(5.4) Subtitle 2**
- Text: `MASTER CONFIGURATION DOSSIER — VAULT EDITION`
- Font: Cormorant Garamond
- Size: 20 pt
- Tracking: +100
- Color: `#C6A861`
- Position: Y ≈ 1.55"

### 6) Main Sigil (Centerpiece)
Use either:
- A custom seal graphic (preferred), or
- A placeholder built from shapes:

**Seal placeholder build (Canva shapes):**
- Outer ring: circle outline, stroke 6 px, `#D4AF37`, 85% opacity
- Inner ring: circle outline, stroke 2 px, `#E1CC87`, 60% opacity
- Laurel wreath: search Canva elements “laurel wreath line art”
- Center monogram text: `ITH` or `HTN`
  - Font: Cinzel
  - Size: 72–96 pt
  - Color: `#D4AF37`

Placement:
- Center horizontally
- Center vertically around Y ≈ 4.40" (slightly above page center)
- Total sigil height: ~4.2" (≈ 38–40% of page height)

### 7) Lower Banner (Classification Block)
Add a low-opacity bar behind text:
- Rectangle across width (inside safe margin)
- Height: ~1.15"
- Fill: black with subtle gradient (or solid `#000000`)
- Opacity: 10–15%

Text (centered, 3 lines):
- `ACCESS LEVEL: Trustee Only — Tier VI Clearance`
- `AUTHORIZED OPERATOR: Isiah-Tarik: Howard, Trustee`
- `CODENAME: Project Orphion — The Speaking Engine of the Trust`

Style:
- Font: Cormorant Garamond
- Size: 16 pt
- Color: `#CBAF72`
- Line spacing: 1.15
- Position: start around Y ≈ 9.55"

### 8) Footer Marking
- Thin divider line above footer: `#C6A861` at 60% opacity
- Footer text (centered):
  - `OFFICIAL TRUST VAULT MARKING`
  - `Edition: Mythic Black-Gold · Volume: Speech Engine Codex, Book I · Status: ACTIVE — Embedded in SintraPrime Core`

Style:
- Font: Cormorant Garamond Italic
- Size: 12–13 pt
- Opacity: 75%
- Position baseline around Y ≈ 10.65"
