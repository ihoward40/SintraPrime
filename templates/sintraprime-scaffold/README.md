# C:\SintraPrime Scaffold (Template)

This is a declarative scaffold for a Windows-local layout.

It is not an installer and does not assert runtime behavior.

## Layout

- `C_SintraPrime/` is a placeholder directory representing `C:\SintraPrime`.
- Populate with local binaries, public keys, and state directories as needed.
