## Purpose

Create stunning, zero-dependency HTML presentations with curated visual styles. Supports SintraPrime pitch decks, investor presentations, and Product Hunt launch materials.

## Key Features

- Single HTML file output (zero dependencies)
- Keyboard, touch, and mouse navigation
- Scroll-triggered animations
- Responsive design with reduced motion support
- PPT-to-web conversion capability

## Usage

```bash
# Generate a presentation
node slides.js --style "Midnight Executive" --input content.md --output deck.html

# Convert PPT to web
node convert.js --input presentation.pptx --style "Swiss Modern"
```
