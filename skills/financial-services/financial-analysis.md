# Financial Analysis Suite Skill

**Category:** Financial Services  
**Source:** github.com/anthropics/financial-services-plugins/financial-analysis  
**Agent:** Financial Analysis Agent  
**Version:** 1.0.0  
**Status:** Active  

## Purpose

Institutional-quality financial modeling including comparable company analysis, DCF valuation, LBO models, and 3-statement financials.

## Commands

| Command | Description |
|---------|-------------|
| `/comps [company]` | Comparable company analysis |
| `/dcf [company]` | DCF valuation model |
| `/lbo [company]` | Leveraged buyout model |
| `/3statements [company]` | Three-statement financial model |
| `/check-deck [file]` | QC presentation for errors |
| `/check-model [file]` | QC financial model for errors |
