# Private Equity Plugin Skill

**Category:** Financial Services  
**Source:** github.com/anthropics/financial-services-plugins/private-equity  
**Agent:** Private Equity Agent  
**Version:** 1.0.0  
**Status:** Active  

## Purpose

Deal sourcing, screening, due diligence, and portfolio monitoring for private equity operations.

## Commands

| Command | Description |
|---------|-------------|
| `/source [criteria]` | Deal sourcing |
| `/screen [deal]` | Deal screening against fund criteria |
| `/dd-checklist [deal]` | Due diligence checklist |
| `/ic-memo [project]` | Investment committee memo |
| `/portfolio-kpi [company]` | Portfolio company KPI dashboard |
| `/returns [deal]` | Returns analysis (IRR, MOIC, DPI) |
| `/value-creation [company]` | Value creation plan |

## ROI Impact

- Critical for trust investment operations and asset management
- Screens 10x more deals with consistent criteria
- Institutional-quality IC memos in hours vs days
