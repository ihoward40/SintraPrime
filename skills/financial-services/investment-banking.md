# Investment Banking Plugin Skill

**Category:** Financial Services  
**Source:** github.com/anthropics/financial-services-plugins/investment-banking  
**Agent:** Investment Banking Agent  
**Version:** 1.0.0  
**Status:** Active  

## Purpose

Sell-side and buy-side M&A support including CIM drafting, pitch decks, and merger modeling.

## Commands

| Command | Description |
|---------|-------------|
| `/cim [company]` | Draft Confidential Information Memorandum |
| `/pitch-deck [topic]` | Create pitch deck |
| `/merger-model [acquirer] [target]` | Build merger model |
| `/buyer-list [company]` | Generate potential buyer list |
| `/deal-tracker [deal]` | Track deal milestones |
| `/teaser [company]` | Draft deal teaser |
| `/process-letter [deal]` | Draft process letter |

## ROI Impact

- Supports SintraPrime productization and enterprise sales
- Reduces CIM drafting time from weeks to days
- Professional-quality pitch materials for investor presentations
